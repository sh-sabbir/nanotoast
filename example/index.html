Meet @iamsabbir/nanotoast

A lightweight, delightful toast notification library for your JavaScript projects.
Supports success, error, warning, info, messages with description, and promise-based toasts.

Usage Examples

Installation

        npm install @iamsabbir/nanotoast
        # or
        yarn add @iamsabbir/nanotoast

Usage

        import toast from "@iamsabbir/nanotoast";
        import "@iamsabbir/nanotoast/dist/styles.css";
        
        toast.success("Hello from NanoToast!");
